# Technical Specification: Shershadow.online

## Tech Stack
-   **Framework**: Next.js 14+ (App Router).
-   **Language**: TypeScript (Strict Mode).
-   **Styling**: Tailwind CSS (JIT Engine).
-   **Animations**: Framer Motion (Scroll-triggered, stagger-loaded).
-   **Deployment**: Vercel (Production Build).

## Design System: Industrial-AI
The visual identity follows a "Hard Logic" industrial aesthetic.

### Typography
-   **Primary (Headers)**: `Syncopate` (Tracking: tight, Case: uppercase).
-   **Secondary (Subheaders/UI)**: `Inter` (Sans-serif, light weights).
-   **Data/Telemetry**: `JetBrains Mono` or standard Monospace.

### Color Palette
-   **Background**: `#0A0A0A` (Deep Carbon).
-   **Primary**: `#D4AF37` (Aesthetic Gold - represents yield/value).
-   **Accent**: `#00F0FF` (Neon Scan - represents diagnostic/UI).
-   **Borders**: `rgba(255, 255, 255, 0.05)` to `0.1` (Subtle structural lines).

### Components Layout
-   **Density**: High (Industrial feel). Vertical padding standardized to `py-16` or `py-24` depending on block significance.
-   **Interaction**: Hover effects with scale and gold glow. Scroll-reveal animations for all data-heavy blocks.

## Feature Architecture
1.  **SFI Test Engine**: 12-question logic mapping to 4 result buckets.
2.  **Shadow Tax Calculator**: Real-time visualization of potential revenue loss.
3.  **Telemetry Dashboard**: Mobile-optimized data visualization of the "Subconscious OS".
